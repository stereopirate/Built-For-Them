## 03 — Adjustable Activity Table

**Status:** 🟡 In Progress

**Purpose:** Accessible activity surface for children using wheelchairs, feeder seats, or positioning equipment. Prioritizes open-front access, adaptive-seat compatibility, and interchangeable tops.

**Commercial equivalent:** Tumble Forms 2 Feeder Seat Tray (Performance Health) — ~$200+. Ours: est. $60–90 in materials.

**Naming rule:** Never position as a generic kids activity table. Always frame around adaptive access — wheelchair clearance, feeder seat fit, positioning equipment compatibility.

### V1 Scope (decided 2026-06-18)

- Freestanding table with basic height adjustment
- Open front, rear-only bracing for wheelchair/adaptive seat access
- Three interchangeable tops sharing one common body cutout and attachment pattern:
  - Sealed wood (daily use)
  - Writable laminate (creative play)
  - Activity insert top (sensory/LEGO)
- Modular accessory attachments deferred to v2

### Key Decisions
