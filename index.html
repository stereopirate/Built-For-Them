Why I started this

I'm a parent who needed this to exist.

I'm raising a disabled child, and I learned quickly how expensive adaptive equipment can be — even for simple things that don't need to be medical grade. Built For Them is my attempt to close that gap: free, practical plans that any parent, grandparent, or maker can build with basic tools and materials from a hardware store.

Affordable

Built from common materials, not specialty medical parts, to keep real costs low.

Made from experience

Every plan starts from a need I've actually run into, not a hypothetical one.

Buildable by anyone

No professional shop required — clear steps, common tools, realistic skill level.

Shared freely

If a plan helps my family, I want it to help yours too, at no cost.
